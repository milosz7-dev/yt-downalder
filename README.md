# YT Downloader v3 🚀

Prosty i szybki downloader wideo z YouTube zbudowany w **Node.js** z wykorzystaniem potężnego silnika **yt-dlp**. Projekt pozwala na pobieranie filmów bezpośrednio przez przeglądarkę na Twój dysk.

## ✨ Funkcje
- Pobieranie wideo w formacie MP4 (obraz + dźwięk).
- Prosty interfejs użytkownika (HTML/CSS).
- Brak zależności od zewnętrznych, płatnych API.
- Działa na Windowsie (lokalnie) oraz Linuxie (hosting).
